# About

![SeDOA_Logo](https://github.com/user-attachments/assets/2bc536a9-1a7e-4b25-9790-3bc1b4e3dc42)

This repo is a place to collect information pertaining to SeDOA ideathons. [SeDOA](https://diamond-open-access.de/en/sedoa/) is the German Service Center for Diamond Open Access, and ideathons are an event format that facilitates collective brainstorming. The focus of the SeDOA ideathons is on innovations around diamond open-access publishing workflows. These innovations might have technological, policy or other elements.

A list of events can be found under the [events folder](https://github.com/sedoa/ideathons/tree/main/events).

Find the issues [here](https://github.com/sedoa/ideathons/issues).
