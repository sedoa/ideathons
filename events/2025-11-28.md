# About

This file assists with the planning, running and documentation of the first SeDOA ideathon, taking place on 28 November 2025.

Die erste Ideenfindungs-Runde ergab eine Reihe von Themen, die wie folgt geclustert wurden.

### Ideen-Cluster
1. Publikationsformat: Enhanced Publishing, dynamische Formate, Granularität
2. KI und Produktionsprozess
3. Medienneutrales Publizieren
4. Shared Infrastructure, One Repository Host
5. KI und Qualitätssicherung, insbes. Peer Review
6. Community Building / Kommunikation
7. Qualität und Reputationsökonomie
